Tech Info

DATASET OVERVIEW

Our model is built using a rich dataset that consists of historical groundwater measurements collected over the past decade from approximately 500 locations. The dataset includes:

  • Station Data: Latitude, longitude, station names, and encoded station IDs for consistent input.
  • Groundwater Surface Elevation (WSE): The key target variable representing water surface elevation, which is predicted.
  • Ground Surface Elevation (WLM_GSE): A related feature representing the elevation of the ground surface.
  • Time-Series Data: Date columns that allow us to identify seasonal trends and long-term patterns in groundwater levels.
  • Cyclic Features: Month encoded using sine and cosine transformations to capture seasonal variations in water levels.
  • Lagging and Rolling Features: Historical values and statistics (mean, median, and standard deviation) based on previous days’ readings
    (e.g., 7-day and 30-day windows), which help in forecasting future trends.

MACHINE LEARNING ALGORITHM USED

To make accurate predictions, we use a combination of three powerful machine learning algorithms: Random Forest, XGBoost, and LSTM. Each of these models contributes uniquely to the prediction process. Random Forest Regression Algorithm: Random Forest is an ensemble learning method that constructs multiple decision trees and merges their results for better accuracy and stability. Why Random Forest?: This method is well-suited for predicting groundwater levels due to its ability to capture complex relationships in the data. It can handle a large number of features and their interactions while preventing overfitting through random sampling. Modeling Approach: We feed the pre-processed dataset with lagging and rolling features into the Random Forest algorithm. The model learns from historical patterns, trends, and fluctuations to predict future groundwater levels.

  • XGBoost (Extreme Gradient Boosting):

    Algorithm:

    : XGBoost is a highly efficient and powerful gradient boosting framework. It focuses on minimizing errors by iteratively improving the model's predictions.

    Why XGBoost:

    XGBoost is ideal for making fast and highly accurate predictions on large datasets. It is particularly effective at identifying nonlinear patterns in the data, making it an excellent choice for groundwater level prediction.

    Modeling Approach:

    XGBoost helps to refine the model by focusing on error minimization. It complements Random Forest’s approach by identifying subtle patterns that may not be captured by decision trees.

  • LSTM (Long Short-Term Memory) Neural Networks :

    Algorithm:

    LSTM is a type of recurrent neural network (RNN) that excels in handling time-series data with long-term dependencies. LSTM can “remember” information over time, which is crucial when dealing with data that spans several years.

    Why LSTM ?:

    Groundwater levels are influenced by long-term trends such as rainfall patterns, seasonal changes, and human activities. LSTM’s ability to remember and learn from long-term dependencies helps capture these trends and forecast water levels more accurately.

    Modeling Approach:

    We use LSTM to model the temporal aspect of the data, accounting for month-wise variations and capturing the long-term effects of environmental factors on groundwater levels.

MODEL INTEGRATION

Our approach integrates the strengths of Random Forest, XGBoost, and LSTM to deliver robust predictions:

  • Pre-Processing: The dataset is first encoded with features such as cyclic month indicators, lagging features, and rolling statistics. Missing values are handled using statistical imputation methods.
  • Feature Engineering: Important features like Ground Surface Elevation (WLM_GSE), Water Surface Elevation (WSE), and rolling statistics are used to capture temporal trends. Cyclic features such as sine and cosine transformations of the month help the model capture seasonal variations.
  • Prediction: Depending on the task, we select the best-performing algorithm or an ensemble approach to combine predictions from multiple models. For time-sensitive predictions,
    LSTM is favored, while for spatial predictions, Random Forest and XGBoost are highly effective.

TECHNICAL DETAILS AND INFRASTRUCTURE

  • Data Storage and Access: The data is stored in a structured format using modern databases and accessed through APIs to ensure seamless and efficient data retrieval.
  • Feature Scaling: Data related to groundwater levels and surface elevations is scaled using robust scaling techniques to prevent outliers from distorting predictions.
  • Sample Weighting: We incorporate sample weighting based on the quality codes for the measurements, giving more importance to high-quality data.
  • Hyperparameter Tuning: The models are tuned using techniques such as grid search to optimize parameters like max_depth, min_samples_split, and learning rates, ensuring that the models generalize well on unseen data.
